# Welcome to your Expo app 👋

This is a placeholder README

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
Functionality:
-Add stocks to watchlist using a search query
-Auto fetches updated price on a user set interval
-Delete stocks from watchlist
-Stock price colour coded relative to previous fetched price


Earnings screen is being pushed in "_layout.tsx" (the calendar symbol)

Stock tick/detail is being pushed in "index.tsx" (line 149)

News screen should mimic the earnings behaviour, but from the stock detail screen (use a newspaper ionicon if it exists)
